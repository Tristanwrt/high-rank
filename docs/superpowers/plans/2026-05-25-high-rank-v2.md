# high-rank.io v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refondre `index.html` du site high-rank.io avec un copy accessible (FR par défaut + EN via switcher) sans jargon SEO, en conservant intégralement la charte visuelle Signal.

**Architecture:** Site HTML statique mono-fichier (déployé sur Hostinger via MCP). L'i18n est entièrement côté client : dictionnaire JS inline + fonction `applyLang()` qui swap `textContent`/`innerHTML`/attributs via `data-i18n*` + détection auto `navigator.language` + override persistant `localStorage`. Pas de routing serveur, pas de build.

**Tech Stack:** HTML5, CSS3, vanilla JS (zéro dépendance). Déploiement via `mcp__hostinger__hosting_deployStaticWebsite`.

**Spec:** [docs/superpowers/specs/2026-05-25-high-rank-v2-design.md](../specs/2026-05-25-high-rank-v2-design.md)

## File Structure

Une seule cible : `index.html` (1224 lignes au moment du plan).

Zones de travail dans le fichier :
- **Lignes 1-21** : `<head>` — title, meta description, OG tags → ajouter `data-i18n` + `data-i18n-attr`.
- **Lignes 22-500** : `<style>` — ajouter règles `.lang-switcher` / `.lang-pill`.
- **Lignes 525-538** : `<nav>` — ajouter le switcher + reformuler les liens.
- **Lignes 539-983** : sections (Hero, Problème, etc.) — ajouter markers + remplacer copy par FR.
- **Lignes 984-1222** : `<script>` — ajouter dict i18n + `applyLang()` + handlers.

**Convention de clés i18n** : `<section>.<element>` (ex: `hero.h1`, `problem.card1.title`, `nav.link.method`). Toutes les clés sont définies dans la même structure plate.

**Convention de marqueurs DOM** :
- `data-i18n="<key>"` → swap `textContent`
- `data-i18n-html="<key>"` → swap `innerHTML` (pour les titres avec `<em>` highlightés)
- `data-i18n-attr="<attr>:<key>;<attr>:<key>"` → swap attributs (ex: `placeholder`, `aria-label`)

**Stratégie de fallback** : Le HTML par défaut contient le copy **français** (la langue dominante du marché). Au load, si JS désactivé ou en panne, le visiteur voit le FR. Le swap vers EN ne se déclenche que si `navigator.language` non-FR ou choix utilisateur stocké.

- [ ] **Step 4 : Ajouter l'i18n dict + `applyLang()` au DÉBUT du `<script>` (juste après ligne 984)**

Insérer ce bloc juste après la ligne `<script>` (ligne 984), AVANT tout le code existant :

```js
// ── i18n ──
const i18n = {
  fr: {
    'nav.why': 'Pourquoi',
    'nav.method': 'Méthode',
    'nav.services': 'Services',
    'nav.about': 'À propos',
    'nav.faq': 'FAQ',
    'nav.cta': 'Mon diagnostic gratuit →'
  },
  en: {
    'nav.why': 'Why',
    'nav.method': 'Method',
    'nav.services': 'Services',
    'nav.about': 'About',
    'nav.faq': 'FAQ',
    'nav.cta': 'Get my free audit →'
  }
};

function applyLang(lang) {
  if (lang !== 'fr' && lang !== 'en') lang = 'fr';
  document.documentElement.lang = lang;
  const dict = i18n[lang];

  // textContent
  document.querySelectorAll('[data-i18n]').forEach(el => {
    const key = el.getAttribute('data-i18n');
    if (dict[key] !== undefined) el.textContent = dict[key];
  });

  // innerHTML (pour titres avec <em> highlight)
  document.querySelectorAll('[data-i18n-html]').forEach(el => {
    const key = el.getAttribute('data-i18n-html');
    if (dict[key] !== undefined) el.innerHTML = dict[key];
  });

  // attributs (placeholder, aria-label, alt, etc.)
  document.querySelectorAll('[data-i18n-attr]').forEach(el => {
    const spec = el.getAttribute('data-i18n-attr');
    spec.split(';').forEach(pair => {
      const idx = pair.indexOf(':');
      if (idx < 0) return;
      const attr = pair.slice(0, idx).trim();
      const key = pair.slice(idx + 1).trim();
      if (dict[key] !== undefined) el.setAttribute(attr, dict[key]);
    });
  });

  // title + meta description
  if (dict['meta.title']) document.title = dict['meta.title'];
  const metaDesc = document.querySelector('meta[name="description"]');
  if (metaDesc && dict['meta.description']) metaDesc.setAttribute('content', dict['meta.description']);

  // switcher UI
  document.querySelectorAll('.lang-pill').forEach(p => {
    const active = p.dataset.lang === lang;
    p.classList.toggle('active', active);
    p.setAttribute('aria-pressed', active ? 'true' : 'false');
  });

  // persist
  try { localStorage.setItem('hr-lang', lang); } catch (e) {}
}

document.addEventListener('DOMContentLoaded', () => {
  let stored = null;
  try { stored = localStorage.getItem('hr-lang'); } catch (e) {}
  const browserFr = (navigator.language || 'fr').toLowerCase().startsWith('fr');
  applyLang(stored || (browserFr ? 'fr' : 'en'));

  document.querySelectorAll('.lang-pill').forEach(p => {
    p.addEventListener('click', () => applyLang(p.dataset.lang));
  });
});
```
